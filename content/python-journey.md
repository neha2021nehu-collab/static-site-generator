# My Python Journey

Learning Python has been one of the most rewarding experiences of my career.
It rewards curiosity — the more you dig, the more elegant it gets.

## Why I Started

I began with web scraping using requests and BeautifulSoup.
Simple tools, but they opened the door to everything that followed.

## What I Am Building Now

Right now I am building a Static Site Generator from scratch.
Not because there are no existing tools, but because building one teaches you
how good tools are designed from the inside out.

### The Architecture

The SSG uses Abstract Base Classes to define plugin contracts.
Every parser, renderer, and theme must fulfill an interface.
Python enforces this at class definition time, not at runtime.

### What I Learned

Metaclasses and ABCs are not magic.
They are just Python being consistent about how classes are objects too.
